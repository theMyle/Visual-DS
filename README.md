# Visual DS

Visual DS is an e-learning platform focused on data structures.

It includes:

- Lessons for learning core data structure concepts
- Simulators for visualizing data structure operations
- Assessments for quiz-based practice and progress checks

Clerk is used as the identity provider for authentication.

## Frontend Stack

- Next.js: App framework and routing for the web client
- React: UI component system and state-driven rendering
- TanStack Query: Server state fetching, caching, and mutations
- Clerk: Authentication and user identity management
- Vercel: Frontend hosting and deployment platform

## Backend Stack

- Golang: Backend API and core service logic
- sqlc: Type-safe SQL query generation for Go
- goose: Database migration management
- Supabase: Managed Postgres database
- Render: Backend hosting and deployment platform
